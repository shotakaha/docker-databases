# docker-mariadb

docker compose MariaDB with Adminer

## Images

- MariaDB: https://hub.docker.com/_/mariadb
- Adminer: https://hub.docker.com/_/adminer

## Usage

```console
// Start services
$ docker compose up -d

// Open adminer via browser
$ open http://localhost:8080

// Quit services
$ docker compose down -v
```
